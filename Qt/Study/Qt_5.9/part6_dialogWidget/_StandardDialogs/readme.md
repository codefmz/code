# _SplashLogin

![image-20230427193255577](./assets/image-20230427193255577.png)

## 功能

1. 实现 Splash 特点的无边框对话框
2. 设计鼠标无边框对话框
3. 使用`QSettings`类存储用户名、密码等信息
4. 用`QCryptographicHash`类进行字符串的加密
5. 根据登录输入状况确定启动主窗口或终止程序进行
6. 无边框窗口移动 `move`函数
